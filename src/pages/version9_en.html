Version 9

  • Property system allowing to create customised visualisations (alliances, groups, ...)

  • Display a label at date change to better see the date.
  • Leap year management.
  • Ability to select a time zone and see all markers in that zone.
  • Possibility to enlarge the time bar in the settings.

  • Keyboard shortcuts :
  • CTRL + S : Save
  • CTRL + Z : Cancel
  • CTRL + Y : Re-establish
  • CTRL + C : Copy
  • CTRL + V : Paste

  • Various bug fixes.
